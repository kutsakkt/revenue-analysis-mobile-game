# revenue-analysis-mobile-game
Анализ динамики выручки и дерева метрик для мобильной игры. Использовался Postgres.

**Цели**:
1) Проанализировать динамику выручки в различных разрезах и понять, есть ли в ней аномалии;
2) Используя дерево метрик, найти причины наблюдаемой динамики.

Данные представляют собой систему из следующих таблиц:
1) Таблица с транзакциями игроков;
2) Таблица с сессиями игроков;
3) Таблица с информацией о том, через какие рекламные каналы мы взаимодействовали с игроком перед тем, как он установил игру;
4) Таблица с характеристиками игроков - страна, платежный сегмент, платформа и т.п.
